# Image Retrieval System
The Project implements Image based search

## Setting up server
Assuming you have all the python dependencies installed
```
cd server
python train_and_build.py
```
It will take some time to build and save the model

## Running the server
```
cd server
python server.py
```

## Setting up client
```
cd client
npm install
```

## Running the client
```
cd client
npm start
```
